# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Professional GitHub repository setup: issue templates, PR template, CI workflow.
- Enhanced `README.md` with dynamic badges, feature grid, and quick-start guides.
- `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and `CHANGELOG.md`.

## [0.1.0] - 2026-05-16

### Added
- Initial release of the **vibecoding** template.
- Next.js 16 with App Router + React 19.
- Tailwind CSS v4 + `tw-animate-css` + shadcn/ui primitives.
- GSAP + ScrollTrigger + ScrollToPlugin pre-registered.
- Lenis smooth scroll provider.
- Three.js + `@react-three/fiber` + `@react-three/drei` + `@react-three/postprocessing`.
- Framer Motion + `@react-spring/web` for declarative UI animation.
- Motion primitives: `FadeIn`, `TextReveal`, `Magnetic`, `Parallax`.
- UI components: `LoadingScreen`, `CustomCursor`.
- Three.js wrappers: `CanvasWrapper`, `PostFX`.
- Hooks: `useGsapScroll`, `useMousePosition`, `useReducedMotion`, `useScrollProgress`.
- Form infrastructure: `react-hook-form` + `zod` + `@hookform/resolvers`.
- i18n ready with `next-intl`.
- Email ready with `resend` + `@react-email/components`.
- Analytics ready with `posthog-js`.
- Utility scripts: `extract-frames.ps1`, `compress-glb.ps1`.
- Strict TypeScript configuration with zero `any` tolerance.
